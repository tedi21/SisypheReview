# Date
Cet objet représente une date. Il dispose diverses méthodes permettant de manipuler une date.

# Référence
## Constructeur Date
*Paramètres*
* None.

*Retour*
* None.

*Description*
* Construit une date.
```
d = new Date(2014, 10, 30);
print(d);
```

## Méthode CurrentDate
*Paramètres*
* None.

*Retour*
* Date : La date courante.

*Description*
* Retourne la date courante.
```
d = new Date().CurrentDate;
print(d);
```

## Méthode Day
*Paramètres*
* None.

*Retour*
* Numeric : le jour du mois.

*Description*
* Retourne le jour du mois.
```
d = new Date().CurrentDate;
print(d.Day);
```

## Méthode Month
*Paramètres*
* None.

*Retour*
* Numeric : le mois.

*Description*
* Retourne le mois.
```
d = new Date().CurrentDate;
print(d.Month);
```

## Méthode Year
*Paramètres*
* None.

*Retour*
* Numeric : l'année.

*Description*
* Retourne l'année.
```
d = new Date().CurrentDate;
print(d.Year);
```

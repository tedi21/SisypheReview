# WordXML
 Classe permettant d'initialiser ou libérer les objets XML.

# Référence
## Méthode Initialize
*Paramètres*
* None.

*Retour*
* None.

*Description*
*  Initialise le parseur XML.
```
xml = new WordXML();
xml.Initialize();
```

## Méthode Terminate
*Paramètres*
* None.

*Retour*
* None.

*Description*
*  Libère les objets XML.
```
xml.Terminate();
xml = null;
```
